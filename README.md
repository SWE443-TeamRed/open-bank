## File Structure

```
open-bank/
├── OpenBankApplication/
│   └── app
│       └── src
│           ├── androidTest
│           ├── main
│           └── test
│   └── gradle/wrapper
├── open-bank/
│   ├── Mockups
│   ├── doc
│   └── src
│       ├── main
│       └── test
├── kanban/
│   ├── Done
│   ├── InProcess
│   ├── Research Documentation
│   └── ToDo
└── scenarios/
```
<br>
OpenBankApplication:    front-end<br>
open-bank:              back-end<br>
kanban:                 task management<br>
scenarios:              story-driven-modeling<br>
